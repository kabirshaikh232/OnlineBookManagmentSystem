# 📚 Online Book Store Management System  
👉 **🔗 Live Demo:** http://www.bookshelf.somee.com/

A web-based application built using **ASP.NET Web Forms and SQL Server** for browsing, searching, and purchasing books online, with admin features to manage inventory and orders.

---

## 🚀 Features

**User**
- Register & Login  
- Browse/Search Books  
- Add to Cart & Place Orders  

**Admin**
- Manage Books (Add/Update/Delete)  
- Manage Orders  
- Dashboard  

---

## 🏗️ Tech Stack
- ASP.NET Web Forms (ASPX)  
- C# (.NET Framework)  
- SQL Server  
- ADO.NET  

---

## ⚙️ Setup
1. Clone repo  
2. Configure SQL Server  
3. Update `Web.config`  
4. Run (F5)

---

## 🎯 Highlights
- 3-Tier Architecture  
- Secure (Parameterized Queries, Sessions)  
- Full E-commerce Flow
